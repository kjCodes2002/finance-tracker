# Expense Tracker Backend

An expense tracking application built using FastAPI and SQLAlchemy for Backend and React + Tailwind for frontend.

## Tech Stack

- FastAPI
- SQLAlchemy
- PostgreSQL
- Pydantic
- React(vite)
- Tailwindcss

## Features

- Wallet management (CRUD)
- Category management (CRUD)
- Transaction management (CRUD)
- Transactions linked to wallets and categories
- User ownership enforced at all routes
- Wallet balance computed from transactions (not stored)
